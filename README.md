## Student Group Project Management System

FSAD-PS35

A web-based platform designed to manage student group projects efficiently by enabling task coordination, progress tracking, collaboration, submission, and grading through role-based access.

## Problem Statement

# FSAD-PS35: Create a web-based platform for managing student group projects.

The system should support:

Task and project assignment

Progress tracking

Collaboration tools

Project submission and evaluation

# Objective

To design and develop a centralized web application that helps teachers manage group projects and students collaborate effectively, track milestones, submit work, and receive feedback.

# Technologies Used

HTML5 – Structure of the application

CSS3 – Styling and responsive design

JavaScript (ES6) – Application logic and interactivity

Browser LocalStorage – Client-side data persistence

 User Roles
# Teacher (Admin)

Assign group projects

Create and manage student groups

Monitor project progress

Review submissions

Grade students

Mark projects as completed

# Student (User)

View assigned projects

Track project milestones

Submit progress reports

Collaborate using group chat

Upload and share files

Submit final project work

## Key Features

Role-based login (Teacher & Student)

Group and project creation

Progress reporting and tracking

Group chat for collaboration

File upload, preview, and download

Project submission with checklist

Grading system with letter grades

Responsive and user-friendly UI
